| Дата конца | Когда кредит должен быть закрыт |
| Статус | Активен/закрыт/просрочен |
| Просрочка | Информация о просрочках |
| Примечание | Дополнительная информация |

Полная история каждого платежа:
| Поле | Содержание |

## 🔧 Техническая Интеграция

### Как Сенсей Использует Google Sheets

1. **При старте бота:**
   - `init_sheets()` создаёт листы если их нет
   - Инициализирует "Кредиты" с начальными данными
   - Инициализирует "История" с заголовками

2. **При упоминании платежа:**
   - Финансовый парсер автоматически:
     - Вызывает `log_payment()` или `log_early_repayment()`
     - Добавляет запись в лист "История"
     - Обновляет остаток в листе "Кредиты"

3. **При запросе информации:**
   - Сенсей может вызвать `get_credit_balance()` для получения текущего баланса
   - Получает реальные данные из Google Sheets

### Пример Использования в Коде

```python
from bot.services.sheets import log_payment, log_early_repayment

# Когда пользователь сообщает "Я заплатил 2.5млн Анорбанку №1"
log_payment(
    credit_name="Анорбанк №1",
    amount=2500000,
    comment="Оплачено через мобильный банк",
    new_balance=15045179  # новый остаток
)

# Когда нужно закрыть часть долга досрочно
log_early_repayment(
    credit_name="AVO карта",
    amount=5000000,
    comment="Выплата для понижения просрочки"
)
```

## 🚀 Развёртывание на Сервере

### Шаг 1: Подготовка Google Cloud
1. Создать Google Таблицу: https://sheets.google.com
2. Создать Service Account в Google Cloud Console
3. Скачать JSON-ключ
4. Поделиться таблицей с email Service Account

### Шаг 2: Загрузка на Сервер
```bash
# В Portainer → контейнер telegrambot:
scp google_credentials.json user@сервер:/app/
```

### Шаг 3: Конфигурация
Добавить в `.env`:
```env
GOOGLE_SHEETS_ID=1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p
GOOGLE_CREDENTIALS_PATH=/app/google_credentials.json
```

### Шаг 4: Установка и Перезагрузка
```bash
pip install google-auth-oauthlib google-auth-httplib2 google-api-python-client
supervisorctl restart telegrambot
```

## 🔍 Возможные Проблемы и Решения

| Проблема | Решение |
|----------|---------|
| "Файл google_credentials.json не найден" | Проверить путь `/app/google_credentials.json` |
| "403 Forbidden" | Проверить что таблица поделена с Service Account |
| "Spreadsheet not found" | Проверить ID таблицы в .env |
| "Сервис недоступен" | Установить зависимости: `pip install google-auth-oauthlib...` |
| Таблица не обновляется | Проверить права доступа Service Account |
