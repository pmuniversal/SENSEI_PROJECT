# СТРУКТУРА ДАННЫХ (актуально с 02.06.2026)

Полная карта проекта: см. STRUCTURE.md в папке CODE.

Код:

/app/main.py        — переходник (запускает bot/main.py)

/app/bot/           — весь код бота
  config.py         — токены и пути (читает .env)
  main.py           — точка входа
  handlers/         — приём сообщений (text, voice, commands)
  services/         — логика (ai, memory, tasks, reminders, backup)
  database/         — db.py (подключение + таблицы)
  utils/            — folders.py, docx.py

Данные:

/app/data/
  memory.db         — база данных
  backups/          — копии базы (раз в час)
  docs/             — сгенерированные DOCX
  voice/            — временные голосовые файлы

Секреты:

/app/.env           — токены (НЕ в git)
/app/.env.example   — шаблон-образец
