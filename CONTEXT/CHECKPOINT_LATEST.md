## КАК ОБНОВЛЯТЬ ФАЙЛЫ НА СЕРВЕРЕ (важно!)

❌ НЕ использовать curl — кеширует GitHub
✅ Использовать python3 urllib:
```bash
python3 -c "
import urllib.request
url = 'https://raw.githubusercontent.com/pmuniversal/SENSEI_PROJECT/main/CODE/bot/services/ФАЙЛ.py'
req = urllib.request.Request(url, headers={'Cache-Control': 'no-cache', 'Pragma': 'no-cache'})
content = urllib.request.urlopen(req).read().decode()
open('/app/bot/services/ФАЙЛ.py', 'w').write(content)
print('OK:', len(content))
"
```
